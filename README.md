# z_tensor

A Python package for tensor operations.

## Installation

```bash
pip install -e .
```

## Development

### Setup

1. Clone the repository
2. Install in development mode with test dependencies:
```bash
pip install -e ".[dev]"
```

### Running Tests

```bash
pytest
```

With coverage:
```bash
pytest --cov=z_tensor --cov-report=html
```

## Usage

```python
import z_tensor

# Your code here
```
